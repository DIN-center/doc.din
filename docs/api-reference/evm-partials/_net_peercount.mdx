# `net_peerCount`


Returns the number of peers currently connected to the client. 

## Parameters

None

## Returns

`peer count`: A hexadecimal of the number of connected peers.

## Example

Replace `<ENDPOINT>` with an API endpoint from your DIN gateway.

### Request


<Tabs>
  <TabItem value="curl">

```bash
curl <ENDPOINT> \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "method": "net_peerCount", "params": [], "id": 1}'
```

  </TabItem>
  <TabItem value="WSS">

```bash
wscat -c <ENDPOINT> -x '{"jsonrpc": "2.0", "method": "net_peerCount", "params": [], "id": 1}'
```

  </TabItem>
</Tabs>
