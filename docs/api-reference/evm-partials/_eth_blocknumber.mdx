# `eth_blockNumber`


Returns the current latest block number. 

## Parameters

None

## Returns

A hexadecimal of an integer representing the current block number the client is on.

## Example

Replace `<ENDPOINT>` with an API endpoint from your DIN gateway.

### Request


<Tabs>
  <TabItem value="curl">

```bash
curl <ENDPOINT> \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "method": "eth_blockNumber", "params": [], "id": 1}'
```

  </TabItem>
  <TabItem value="WSS">

```bash
wscat -c <ENDPOINT> -x '{"jsonrpc": "2.0", "method": "eth_blockNumber", "params": [], "id": 1}'
```

  </TabItem>

</Tabs>

### Response


<Tabs>
  <TabItem value="JSON">

```js
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": "0x65a8db"
}
```

  </TabItem>
</Tabs>

:::tip Convert hexadecimal to integer
On macOS, you can use `printf "%d\n" 0x65a8db` to convert the above hexadecimal number into an integer number.
:::
