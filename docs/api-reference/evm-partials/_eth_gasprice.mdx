# `eth_gasPrice`


Returns the current gas price in wei. 

## Parameters

None.

## Returns

A hexadecimal equivalent of an integer representing the current gas price in wei.

## Example

Replace `<ENDPOINT>` with an API endpoint from your DIN gateway.

### Request


<Tabs>
  <TabItem value="curl">

```bash
curl <ENDPOINT> \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "method": "eth_gasPrice", "params": [], "id": 1}'
```

  </TabItem>
  <TabItem value="WSS">

```bash
wscat -c <ENDPOINT> -x '{"jsonrpc": "2.0", "method": "eth_gasPrice", "params": [], "id": 1}'
```

  </TabItem>
</Tabs>

<Tabs>
