# `web3_clientversion`


Returns the current client version. 

## Parameters

None.

## Returns

A string representing the current client version.

## Example

Replace `<ENDPOINT>` with an API endpoint from your DIN gateway.

### Request


<Tabs>
  <TabItem value="curl">

```bash
curl <ENDPOINT> \
  -X POST \
  -H "Content-Type: application/json" \
  -d '{"jsonrpc": "2.0", "method": "web3_clientVersion", "params": [], "id": 1}'
```

  </TabItem>
  <TabItem value="WSS">

```bash
wscat -c <ENDPOINT> -x '{"jsonrpc": "2.0", "method": "web3_clientVersion", "params": [], "id": 1}'
```

  </TabItem>
</Tabs>
